**主要内容**：
- 视频背景展示

- 注册表单（邮箱、密码）
- 表单切换（登录/注册）
- 用户认证逻辑
- 错误处理和提示

1. 页面布局和样式（玻璃拟态设计）
2. Supabase 客户端初始化
3. 用户注册功能
4. 用户登录功能
5. 登录状态检查
6. 页面跳转逻辑
7. 错误处理和提示

### 3.2 app.html（主应用页面）

**功能概述**：核心功能页面，包含遗物创建、录制、管理等功能

**主要内容**：
- 用户导航和退出
- 联系人管理（添加、查看、选择）
- 音视频录制（预览、开始、停止）
- 遗物创建（标题、内容、阈值设置）
- 遗物列表展示
- 遗物播放和删除

**核心代码模块**：
1. 页面布局和样式
2. Supabase 客户端初始化
3. 用户登录状态检查
4. 联系人管理功能
5. 音视频录制功能

## 五、数据库表结构

### 5.1 profiles（用户表）

| 字段名 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | UUID | PRIMARY KEY | 用户唯一标识 |
| email | TEXT | UNIQUE NOT NULL | 用户邮箱 |
| created_at | TIMESTAMP | DEFAULT now() | 创建时间 |
| last_login | TIMESTAMP | - | 最后登录时间 |

### 5.2 contacts（联系人表）

| 字段名 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | UUID | PRIMARY KEY | 联系人唯一标识 |
| user_id | UUID | FOREIGN KEY | 所属用户ID |
| name | TEXT | NOT NULL | 联系人姓名 |
| email | TEXT | - | 联系人邮箱 |
| phone | TEXT | - | 联系人电话 |
| created_at | TIMESTAMP | DEFAULT now() | 创建时间 |

### 5.3 relics（遗物表）

| 字段名 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | UUID | PRIMARY KEY | 遗物唯一标识 |
| user_id | UUID | FOREIGN KEY | 创建者ID |
| title | TEXT | NOT NULL | 遗物标题 |
| content | TEXT | - | 遗物内容 |
| silence_threshold | INTEGER | NOT NULL | 沉默阈值（天数） |
| recipient_id | UUID | FOREIGN KEY | 接收人ID |
| anonymous_send | BOOLEAN | DEFAULT false | 是否匿名发送 |
| video_url | TEXT | - | 视频文件URL |
| audio_url | TEXT | - | 音频文件URL |
| status | TEXT | DEFAULT 'pending' | 发送状态 |
| created_at | TIMESTAMP | DEFAULT now() | 创建时间 |
| sent_at | TIMESTAMP | - | 发送时间 |

## 七、代码原创性声明

本项目的所有代码均为原创开发，未使用任何第三方开源代码或库，具体说明如下：

1. **前端代码**：所有 HTML、CSS、JavaScript 代码均为自主编写
2. **后端服务**：使用 Supabase BaaS 平台，未编写后端代码
3. **第三方依赖**：仅使用浏览器原生 API 和 Supabase SDK
4. **资源文件**：hhzz.mp4、hhzz.png、rad.png 均为自主制作或授权使用

本项目不存在任何代码抄袭或侵权行为，所有代码的知识产权归开发者所有。
