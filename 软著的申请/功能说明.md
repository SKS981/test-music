## 二、功能详细说明

### 2.1 用户认证模块

#### 2.1.1 用户注册
- **功能描述**：用户通过邮箱和密码进行注册
- **输入**：邮箱地址、密码
- **输出**：用户账户创建成功/失败提示
- **流程**：
  1. 用户填写注册表单
  2. 系统验证邮箱格式和密码强度
  3. 调用 Supabase Auth API 创建用户
  4. 发送邮箱验证链接
  5. 用户点击验证链接完成注册

#### 2.1.2 用户登录
- **功能描述**：用户通过邮箱和密码进行登录
- **输入**：邮箱地址、密码
- **输出**：登录成功/失败提示，跳转到主页面
- **流程**：
  1. 用户填写登录表单
  2. 系统验证账号密码
  3. 调用 Supabase Auth API 进行身份验证
  4. 验证成功后设置登录状态
  5. 跳转到主应用页面

#### 2.1.3 会话管理
- **功能描述**：管理用户登录状态
- **功能点**：
  - 自动检测登录状态
  - 未登录用户自动重定向到登录页
  - 登录状态持久化存储
  - 安全退出登录

---

### 2.2 联系人管理模块

#### 2.2.1 添加联系人
- **功能描述**：用户添加新的联系人
- **输入**：联系人姓名、邮箱、电话
- **输出**：联系人添加成功/失败提示
- **流程**：
  1. 用户填写联系人信息表单
  2. 系统验证输入格式
  3. 将联系人信息保存到数据库
  4. 刷新联系人列表

#### 2.2.2 联系人列表
- **功能描述**：展示用户的所有联系人
- **输出**：联系人卡片列表
- **展示内容**：
  - 联系人姓名
  - 联系人邮箱
  - 联系人电话
  - 选择按钮

#### 2.2.3 联系人选择
- **功能描述**：在创建遗物时选择接收人
- **交互**：点击选择按钮将联系人添加到遗物接收列表

---

### 2.3 音视频录制模块

#### 2.3.1 媒体设备调用
- **功能描述**：获取摄像头和麦克风权限
- **流程**：
  1. 用户点击录制按钮
  2. 系统请求媒体设备权限
  3. 获取摄像头和麦克风流
  4. 显示视频预览

#### 2.3.2 视频预览
- **功能描述**：实时显示摄像头画面
- **特点**：
  - 实时预览
  - 支持切换前后摄像头
  - 视频尺寸自适应

#### 2.3.3 录制控制
- **功能描述**：控制音视频录制的开始和停止
- **功能点**：
  - 开始录制
  - 停止录制
  - 暂停录制（可选）
  - 录制时长显示

#### 2.3.4 文件上传
- **功能描述**：将录制的音视频文件上传到云端存储
- **流程**：
  1. 录制完成后生成 Blob 对象
  2. 创建唯一文件名（UUID + 时间戳）
  3. 上传到 Supabase Storage
  4. 获取文件访问链接

---

### 2.4 遗物管理模块

#### 2.4.1 创建遗物
- **功能描述**：创建新的遗物记录
- **输入**：
  - 遗物标题
  - 遗物内容（文字描述）
  - 沉默阈值（年、月、日）
  - 接收人（联系人）
  - 匿名发送选项
  - 音视频文件
- **输出**：遗物创建成功/失败提示
- **流程**：
  1. 用户填写遗物信息
  2. 选择接收联系人
  3. 设置沉默阈值
  4. 录制或上传音视频
  5. 保存遗物到数据库

#### 2.4.2 遗物列表
- **功能描述**：展示用户创建的所有遗物
- **展示内容**：
  - 遗物标题
  - 创建时间
  - 沉默阈值
  - 接收人信息
  - 状态（待发送/已发送）
  - 操作按钮（播放、删除）

#### 2.4.3 遗物播放
- **功能描述**：播放遗物的音视频内容
- **功能点**：
  - 获取文件访问链接
  - 创建视频/音频播放器
  - 播放控制（播放、暂停、音量调节）

#### 2.4.4 删除遗物
- **功能描述**：删除指定的遗物
- **流程**：
  1. 用户点击删除按钮
  2. 确认删除操作
  3. 从数据库删除遗物记录
  4. 从存储删除音视频文件

---

### 2.5 自动发送模块

#### 2.5.1 沉默阈值设置
- **功能描述**：设置遗物自动发送的时间条件
- **输入**：年、月、日
- **计算逻辑**：
  - 总天数 = 年 × 365 + 月 × 30 + 日
  - 发送时间 = 创建时间 + 总天数

#### 2.5.2 定时检查机制
- **功能描述**：定期检查遗物是否达到发送条件
- **检查频率**：页面加载时检查，可设置定时任务
- **检查逻辑**：
  1. 获取当前时间
  2. 遍历所有待发送的遗物
  3. 判断是否达到沉默阈值
  4. 触发发送逻辑

#### 2.5.3 邮件/短信发送
- **功能描述**：向接收人发送遗物通知
- **发送优先级**：邮箱 > 电话 > 无发送
- **邮件内容**：
  - 遗物标题
  - 遗物内容摘要
  - 音视频链接
  - 发送者信息（匿名时隐藏）

## 三、功能流程图

### 3.1 用户登录流程
```
用户访问首页 → 检查登录状态 → 已登录→进入主页面
                          ↓ 未登录
                    显示登录/注册表单 → 用户提交 → 验证通过 → 设置登录状态 → 进入主页面

```
进入创建页面 → 填写标题和内容 → 设置沉默阈值 → 选择接收人
              ↓                                    ↓
         录制音视频                              上传完成
              ↓                                    ↓
         上传到存储                              保存到数据库
              ↓                                    ↓
         创建遗物记录 ←─────────────────────────────

### 3.3 自动发送流程
```
检查遗物列表 → 判断是否达到沉默阈值 → 是→获取接收人信息
              ↓ 否                           ↓
         跳过                              判断联系方式
                                            ↓
                                   有邮箱→发送邮件
                                   ↓ 无邮箱
                              有电话→发送短信
                              ↓ 无电话
                         记录发送失败状态
```

## 四、核心业务规则

### 4.1 沉默阈值计算规则
- 年：按365天计算
- 月：按30天计算
- 日：按实际天数计算
- 总阈值天数 = 年 × 365 + 月 × 30 + 日

### 4.2 发送优先级规则
1. 优先使用邮箱发送
2. 若无邮箱，尝试使用电话发送
3. 若无联系方式，记录发送失败状态

### 4.3 匿名发送规则
- 匿名发送时，隐藏发送者信息
- 邮件/短信中不包含发送者的姓名和联系方式
- 接收者无法回复发送者

### 4.4 文件命名规则
- 文件名格式：UUID/时间戳.扩展名
- 示例：EAF2822C-6FC3-4D5E-8808-E7354FA2F2E1/1774008304720.webm

## 五、数据存储说明

### 5.1 用户数据
- 用户ID（UUID）
- 邮箱地址
- 创建时间
- 最后登录时间

### 5.2 联系人数据
- 联系人ID（UUID）
- 用户ID（外键）
- 姓名
- 邮箱
- 电话
- 创建时间

### 5.3 遗物数据
- 遗物ID（UUID）
- 用户ID（外键）
- 标题
- 内容
- 沉默阈值（天数）
- 创建时间
- 发送状态
- 发送时间
- 匿名发送标记
- 视频URL
- 音频URL

### 5.4 文件存储
- 存储位置：Supabase Storage
- 存储桶名称：recordings
- 文件类型：webm（视频）、webm（音频）
- 访问方式：Signed URLs（安全访问）
