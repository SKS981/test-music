## 二、项目简介

### 2.1 项目背景
"时光留声"是一款创新的Web应用，旨在帮助用户记录和留存珍贵的音视频"遗物"，并在设定的沉默阈值后自动发送给指定的联系人。

### 2.2 核心概念
- **遗物(Realm)**：用户录制的音视频内容，可设定沉默阈值
- **沉默阈值**：用户设定的时间周期，超过该时间未更新则自动发送
- **联系人(Contact)**：遗物的接收人，支持邮箱和电话联系方式

### 2.3 应用场景
1. **情感表达**：记录重要时刻，在特定时间后自动发送给亲友
2. **遗产规划**：提前准备重要信息，在长时间沉默后自动传递
3. **时光胶囊**：创建时间胶囊，设定未来某个时间点自动打开
4. **远程沟通**：在无法亲自沟通的情况下，通过预先录制的内容传递信息

### 2.4 主要功能
1. 用户注册与登录
2. 联系人管理
3. 音视频录制
4. 遗物创建与管理
5. 自动发送机制
6. 匿名发送功能

## 三、技术架构

### 3.1 前端技术栈
- **HTML5**：页面结构
- **CSS3**：样式设计（玻璃拟态风格）
- **JavaScript**：交互逻辑
- **MediaRecorder API**：音视频录制
- **Navigator.mediaDevices API**：获取媒体设备权限

### 3.2 后端技术栈
- **Supabase**：BaaS平台
  - Authentication：用户认证
  - Database：数据存储
  - Storage：文件存储

### 3.3 部署架构
- **前端部署**：Vercel / GitHub Pages
- **后端服务**：Supabase Cloud
- **域名**：自定义域名（如 zqdssg.xyz）
